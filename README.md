This repository contains computational scripts for analysis of small cell lung cancer tumor (SCLC) and CTC single cell RNA-seq data.

# SCLC_DLL3_scRNA_cohortB.Rmd

analysis for scRNA-seq data from ten SCLC tumors collected at MGH.

# SCLC_DLL3_scRNA_cohortC.Rmd

analysis for scRNA-seq data from 19 SCLC tumors downloaded from https://cellxgene.cziscience.com/collections/62e8f058-9c37-48bc-9200-e767f318a8ec.
